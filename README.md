 ## 1.  프로그램 소개 : 
 자신 예산과 지출 관리하는
## 2. 사용 기술 : 
 C# WinForm, MySql
## 3. 주요 기능 설명 : 
비회원, 회원 구분과 일정 내용과 금액 저장하고 저장 되어 있는 것에 수정에서 내용, 금액을 수정 가능하면, 삭제는 1개만 했도 되고 여러개도 가능한다 예산 얼마인지 저장 가능하면 +로 원래 예산에서 추가 가능하면 -로 원래 예산에서 추가 가능한다 그리고 월로 계산했어 지출 총 금액과 그 달에
  가장 많이 사용 할 날 표시와 내 예산보다 지출금액이 많을면 경고가 되어 있고 예산 안에 사용 중 이라면 정상이라고 경고가 오지 않는다
## 4. 실행 방법 :
처음 실행하면 home화면이 나오는데 로그인, 회원가입, 비회원이 있는데 비회원으로 가면 로그인 할 필요는 없지만 회원에 비해 기능이 별로 없다. 회원가입을 먼저 했야지 로그인 가능했어 회원가입를 먼저 하고 로그인에서 로그인 하면 회원에서 창이 나오는데 예산을 저장하고 +,-로 더하는게 가능하고 
 뺏는 것도 가능한다 그리고 달력에 날짜를 클릭하고 내용, 지출 금액 추가 하면 listView에서 표시가 되면 왼쪽에 체크박스를 클릭하고 수정에서 내용과, 지출금액 수정이 가능하고 체크박스로 하나만 선택 했도 되고 여러개 선택했어 삭제를 하난만 했도 되고 여러개도 가능한다.
## 5. 개선 하고 싶은 점 : 
한달 기준으로만 예산, 지출 금액 확인 하지 않고 목록에 따라서 한달 기준 일주일 기준 1년 기준 여러 가지에 따라서 확인이 바로 가능하게 하면 좋을 것 같다. 그리고 회원가입 할 때 DB에 저장되는데 비밀번호가 보여서 보이지 않게 하기 보안을 위해서
